Vos clés API

Documentation API

Intégrez l'unboxing sécurisé SafeTrade dans votre e-commerce. Toutes les requêtes nécessitent votre clé API.

Authorization: Bearer sk_live_xxxxxxxxxxxx...

Authentification

POST /api/b2b/orders Créer une commande

Crée une commande et génère un code de vérification à envoyer à votre client.

curl -X POST https://votre-domaine.com/api/b2b/orders \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "external_order_id": "ORDER-123",
    "customer_email": "client@example.com",
    "customer_name": "Jean Dupont",
    "product_name": "Air Max 90 Blanc T42",
    "order_amount": 129.99,
    "currency": "EUR"
  }'
// Réponse
{
  "order_id": 42,
  "external_order_id": "ORDER-123",
  "verification_code": "A3F9",
  "unboxing_url": "https://votre-domaine.com/unboxing-customer.html?code=A3F9&order=42",
  "message": "Commande créée. Partagez le verification_code avec votre client."
}
GET /api/b2b/orders Lister les commandes

Récupère la liste de vos commandes. Filtrage par statut disponible.

curl "https://votre-domaine.com/api/b2b/orders?status=unboxing_submitted&page=1" \
  -H "Authorization: Bearer sk_live_..."
GET /api/b2b/orders/:id Détail commande 
curl "https://votre-domaine.com/api/b2b/orders/42" \
  -H "Authorization: Bearer sk_live_..."
PATCH /api/b2b/orders/:id/ship Marquer expédiée 
curl -X PATCH https://votre-domaine.com/api/b2b/orders/42/ship \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{"tracking_number": "FR700000001", "carrier": "Colissimo"}'
PATCH /api/b2b/orders/:id/deliver Confirmer livraison (ouvre 48h unboxing) 
curl -X PATCH https://votre-domaine.com/api/b2b/orders/42/deliver \
  -H "Authorization: Bearer sk_live_..."
PATCH /api/b2b/orders/:id/resolve Résoudre (compléter ou rembourser) 
curl -X PATCH https://votre-domaine.com/api/b2b/orders/42/resolve \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{"resolution": "completed"}'   // ou "refunded"
GET /api/b2b/stats Statistiques entreprise 
curl "https://votre-domaine.com/api/b2b/stats" \
  -H "Authorization: Bearer sk_live_..."

Exemple JavaScript

// Créer une commande depuis votre backend Node.js
const res = await fetch('https://votre-domaine.com/api/b2b/orders', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer sk_live_...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    external_order_id: order.id,
    customer_email: customer.email,
    product_name: product.name,
    order_amount: product.price
  })
});
const { verification_code, unboxing_url } = await res.json();

// Envoyez unboxing_url par email à votre client
await sendEmail(customer.email, {
  subject: 'Votre commande — Enregistrez votre unboxing',
  body: `Cliquez ici pour démarrer votre unboxing: ${unboxing_url}`
});
