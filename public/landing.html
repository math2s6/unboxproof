Comment ça marche

Sécurisé en 4 étapes simples

De la commande à la validation, SafeTrade automatise toute la chaîne de confiance entre vous et vos clients.

→
🛒
1

Dès qu'un client commande, SafeTrade génère automatiquement un code unique cryptographique lié à cette commande.

Intégration

Intégrez en 3 lignes de code

Une API REST simple et bien documentée. Commencez en quelques minutes, pas en quelques semaines.

Integration exemple — Node.js 
// 1. Créer une commande avec code de vérification
const response = await fetch('https://api.safetrade.io/b2b/orders', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer sk_live_votre_cle' },
  body: JSON.stringify({
    external_order_id: 'CMD-12345',
    customer_email: 'client@example.com',
    product_name: 'Nike Air Max 90',
    order_amount: 120.00
  })
});

const order = await response.json();
// → Envoyez order.unboxing_url à votre client par email
console.log(order.verification_code); // "A3F7C2B1"
console.log(order.unboxing_url);    // "https://safetrade.io/unboxing?order=..."

✅ Réponse : { order_id: 1247, verification_code: "A3F7C2B1", unboxing_url: "https://..." }
