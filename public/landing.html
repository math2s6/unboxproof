Intégration

Intégrez en 3 lignes de code

Une API REST simple et bien documentée. Compatible avec tous les langages et plateformes.

Exemple d'intégration — Node.js 
// 1. Créer une commande avec code de vérification
const response = await fetch('https://api.unboxproof.io/b2b/orders', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer sk_live_votre_cle' },
  body: JSON.stringify({
    external_order_id: 'CMD-12345',
    customer_email: 'client@example.com',
    product_name: 'Sneakers Pro X',
    order_amount: 149.00
  })
});

const order = await response.json();
// → Envoyez order.unboxing_url à votre client par email
console.log(order.verification_code); // "A3F7C2B1"
console.log(order.unboxing_url);    // "https://unboxproof.io/unboxing?order=..."

✅ Réponse : { order_id: 342, verification_code: "A3F7C2B1", unboxing_url: "https://..." }
